标题

用一到六个 ‘#’ 加一个空格接标题内容

字体

星号或下划线包裹内容，单是斜体，双是粗体，三是粗斜体

写法 效果
*这是斜体* 这是斜体
_这是斜体_ 这是斜体
*这是粗体* 这是粗体
_这是粗体_ 这是粗体
*这是粗斜体* 这是粗斜体
_这是粗斜体_ 这是粗斜体
~~这是删除线~~ 这是删除线
<u>这是下划线 这是下划线

表格

用 ‘|’ 字符分隔内容即可

ctrl+enter自动添加新行

换行

  • 两句话之间加一个空行
  • 如果你在编辑的时候，想让一行文字在显示的时候换行，就在中间加<br/>
